# Latin Mass HTML Site

This is a starter project for a bilingual Latin Mass website built with `.html` pages.

## What is included

- `index.html` - homepage and project overview.
- `parts/order-of-mass.html` - expanded Ordinary with fuller Latin/English text and audio clips.
- `parts/seasonal-propers.html` - Advent, Lent, Easter, and saints-day sample propers.
- `parts/pronunciation-guide.html` - Ecclesiastical Latin pronunciation rules and practice drills.
- `parts/symbolism-guide.html` - explanations of symbols, gestures, and liturgical signs.
- `parts/glossary.html` - core terms and definitions.
- `parts/sources.html` - citation keys and links for text, translation, and audio sources.
- `404.html` - custom not-found page for GitHub Pages routing.
- `assets/css/styles.css` - shared design system and page styles.
- `assets/js/site.js` - shared navigation and page enhancements.
- `IMAGE_SOURCES.md` - open-license image credits and license links.
- `.htaccess` - sets `index.html` as the default document.
